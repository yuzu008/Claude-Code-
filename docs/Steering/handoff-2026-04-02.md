## アーキテクチャ概要

```
3タブ構成（useState で切り替え、ルーティングなし）
├── ガイドタブ: 左ツリー + 右説明パネル
├── ファイル作成タブ: カード選択 → ウィザード（左フォーム + 右ガイド）
└── 概念説明タブ: 左メニュー + 右詳細
```

### データ駆動
- `src/data/treeData.ts` — ツリー構造（TreeNode 型）
- `src/data/explanations.ts` — 各ノードの説明（Explanation 型、IDで紐付け）
- `src/data/concepts.ts` — 概念説明（Concept 型）

コンテンツ追加は **データファイルにオブジェクトを追加するだけ** でUIに反映される。

### 主要コンポーネント
| コンポーネント | 役割 |
|---|---|
| `TreeView` / `TreeNodeItem` | 再帰ツリー描画 |
| `ExplanationPanel` | ツリーノードの説明表示 |
| `WizardLayout` | ウィザード共通UI（ステップ進行 + 右ガイド） |
| `OutputPanel` | 生成結果のコピー/ダウンロード |
| `ConceptsPage` / `ConceptDetail` | 概念説明の左メニュー + 右詳細 |
